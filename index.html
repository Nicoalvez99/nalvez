Prolinko

Prolinko es un sitio web desarrollado en Laravel, PHP y Tailwind CSS, con base de datos MySQL. Especializado en la gestión financiera familiar, calcula presupuestos, organiza ingresos, gastos, ahorros y deudas. Permite vincular cuentas entre usuarios y garantiza seguridad mediante cifrado de datos para una experiencia confiable y protegida.
